
# 5-Day Weather Forecast



* Node.js 9.8.0 and above

## Run
```
git clone
cd weather-foecast or whatever is the folder name.
npm i
```


## Start the dev server
```

npm run start:dev

```
localhost:8080

## Build
```

npm run build

```


* Running the build bundles all your updates to ```bundle.js``` and ```bundle.css``` in dist folder.

Technology used

* React.js
* Redux
* JavaScript (ES6)
* HTML5
* SASS
