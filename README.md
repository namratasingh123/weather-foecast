
# 5-Day Weather Forecast



* Node.js 9.8.0 and above

## Run
```
git clone
cd react-weather-task
npm i
```


## Start the dev server
```

npm run start:dev

```

## Build
```

npm run build

```


* Running the build bundles all your updates to ```bundle.js``` and ```bundle.css``` in dist folder.

Technology used

* React.js
* Redux
* JavaScript (ES6)
* HTML5
* SASS
